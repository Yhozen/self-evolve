# Runs Index

This section is reserved for per-run artifacts.

Use a dated directory such as `docs/runs/2026-04-15-some-run/` when a run needs durable evidence.

Typical contents may include:

- a run brief based on `../templates/run-brief-template.md`
- an evaluation report based on `../templates/evaluation-report-template.md`
- notes about why the run was kept, discarded, or decomposed

Keep historical run evidence here. Do not use this section as the canonical source of current truth.
