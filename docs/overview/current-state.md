- `AGENTS.md` defines the root repository guidance for autonomous runs.
- `docs/` now contains the initial wiki skeleton, including blueprints, evaluation guidance, templates, and decisions.
- The repository is using plain markdown first and keeping the docs tree tool-agnostic.

## Immediate Use

Even in bootstrap form, the wiki already provides:

- a stable home for project knowledge
- a place to record decisions and run evidence
- workflow scaffolding for future autonomous runs
